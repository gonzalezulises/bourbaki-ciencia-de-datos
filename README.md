# Bourbaki - Track para la Ciencia de Datos

> 49 Semanas | 49 Casos de uso reales

Repositorio personal para el programa completo del **Colegio de Matematicas Bourbaki** - Track de Ciencia de Datos & AI.

## Estructura del Track

| # | Curso | Nivel | Semanas |
|---|-------|-------|---------|
| 01 | [Introduccion a Python](./01-introduccion-a-python) | Propedeutico | - |
| 02 | [ML & AI For the Working Analyst](./02-ml-ai-for-the-working-analyst) | Profesional | 9 |
| 03 | [Matematicas para la Ciencia de Datos](./03-matematicas-para-la-ciencia-de-datos) | Practicante/Avanzado | 12 |

## Objetivos

- **A.** Dominar Python para procesar fuentes de informacion en distintos formatos y volumenes, con enfasis en funciones y librerias para escalar soluciones data-driven.
- **B.** Familiarizarse con los conceptos matematicos detras de los algoritmos y modelos de Machine Learning: seleccion de algoritmos, calibrado de modelos y evaluacion con distintas metricas.
- **C.** Aplicar conocimientos en casos de uso con datos reales y problematicas comunes en la industria.

## Evaluacion

- Test semanal por tema
- Examen practico al final de cada bloque (3 bloques por curso)
- Examen teorico simulando entrevista laboral
- Proyecto final con acompanamiento de profesores
